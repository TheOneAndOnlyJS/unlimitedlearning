Square and Square roots



Answer:


To find the square root of 53 we can easy do this without actual multiplication:

53 = (50 + 3)²

Now using distributive property we can solve this:
53 = (50 + 3)² =

50x(50 + 3) + 3x(3 + 50) =

50² + 50 x 3 + 3 x 50 + 3² =

2500 + 150 + 150 + 9 = 2809


Therefore 2809 is the square of 53
